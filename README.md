## Quick Start

```bash
pip install nira-market-intel
```

Add to your Claude Code MCP config (`~/.claude.json`):
```json
{
  "mcpServers": {
    "nira-market-intel": {
      "command": "python",
      "args": ["-m", "nira_market_intel"]
    }
  }
}
```
